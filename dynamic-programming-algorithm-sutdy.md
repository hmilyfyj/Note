# 描述

> 给定一组物品，每种物品都有自己的重量和价格，在**限定的总重量**内，我们如何选择，才能使得物品的总价格最高。

# 解决思想

设有n个物品，背包的重量为`w`，`C[i][w]`为最优解。`v[i]`为第i个商品的价值，`w[i]` 为其重量。

首先我们考虑一个问题：我们从第一件商品开始向背包里放置商品，通过不停的拿取 、组合从而获得了最大价值`C[i][w]`时，最后一个商品`(i=n`)只有两种状态，要么放、要么不放（废话）：

| 商品i的状态	 | C[i][w]     |
|:--------:| -------------:|
| 放时：| w[i] +  C[i - 1][w - w[i]] |
| 不放：| C[i - 1][w] |
